## Title recommendation

RaumBaller

## Short description

A retro portrait space shooter: pick a ship, dodge enemy waves, and survive four levels in your browser.

## Long description

RaumBaller is an old-school scrolling shoot 'em up built around quick arcade runs. Choose one of three ships, steer by tapping or pointing where you want to go, and survive waves of enemy ships across four levels.

Your ship shoots automatically, so the focus stays on movement: dodge bullets, ram only when you have to, grab health pickups, and destroy enemies to raise your score. As your score climbs, your ship's cannons upgrade automatically.

This web version runs directly in the browser with the same Java gameplay code as the Android version, plus desktop-friendly keyboard movement.

## Controls

- Select ship / steer: tap, click, or hold pointer position
- Move: WASD or arrow keys
- Shoot: automatic
- Extra shoot input: Space

## Features

- Browser build with no install
- Original pixel-art assets and sounds
- Three playable ships with different strengths
- Four levels of enemy waves
- Score-based cannon upgrades
- Health pickups
- Level-complete, game-over, and victory flows
- Responsive portrait canvas that keeps the original 9:16 aspect ratio
- Playable with touch, mouse, or keyboard
